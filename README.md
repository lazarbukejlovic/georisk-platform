# GeoRisk

GeoRisk is a React and TypeScript geopolitical risk intelligence platform for conflict monitoring, market impact analysis, historical context, and simulated humanitarian relief tracking in one commercial-grade interface.

## Product Scope

- Global conflict intelligence dashboard
- Cinematic map with regional risk markers and conflict drill-downs
- Situation feed with severity and source context
- Market impact terminal with risk-linked asset monitoring
- Historical archive for conflict comparison and lessons learned
- Humanitarian Relief Tracker with simulated campaign funding visibility
- Reports and settings surfaces kept mocked but integration-ready

## Transparency

- The Humanitarian Relief Tracker is a simulated demo module.
- No real funds are collected or processed.
- Data displayed in the platform is mocked for product demonstration and integration planning.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Query
- React Router
- Vitest
